# Levli Validation 1

גרסה ממוקדת לאימות נתונים בלבד.

## מה היא עושה
- בודקת AAPL, MSFT ו-NVDA.
- מציגה את הערכים עצמם, לא רק ✅/❌.
- מציגה P/E מול Forward P/E, ROE, ROIC, Quick Ratio, Gross Margin, Profit Margin ונתוני Growth.
- מציגה בנפרד איזה endpoint של FMP הצליח ואיזה החזיר 402/403.
- שגיאה ב-endpoint אחד אינה מפילה את כל המניה.

## מה לא השתנה
- חוקי הסינון.
- Levli Score.
- אין פיצ'רים חדשים.

## העלאה
העלה את הקבצים שבתוך התיקייה לשורש ה-Repository. אין להעלות `__pycache__`.
