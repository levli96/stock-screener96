# Levli Beta 1.2

גרסה ממוקדת שממשיכה את Beta 1.1 בלי לשנות את חוקי הסינון.

## מה חדש
- ברירת מחדל: סריקה של 10 מניות.
- טבלת **בדיקת תנאי החובה** לכל מניה.
- ניתן לראות בדיוק אילו תנאים עברו ואילו נכשלו.
- הטבלה הראשית עדיין מציגה רק מניות שעברו את כל תנאי החובה.
- ללא pandas, openpyxl או cache, כדי לשמור על יציבות ב-Streamlit.

העלה את הקבצים שבתיקייה זו לשורש ה-repository והחלף את הקבצים הקיימים. לאחר Commit, בצע Reboot ב-Streamlit.
