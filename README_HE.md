# Levli v0.5

גרסה מודולרית ויציבה יותר של Levli.

## מה תוקן

- שימוש בנתיבי FMP החדשים תחת `/stable`.
- בדיקת API Key נפרדת לפני הסריקה.
- הודעות ברורות עבור 401, 403, 429 ושגיאות תקשורת.
- הפרדה לקבצים: API, סינון, Growth ו-Levli Score.
- אין עוד טקסט ברירת מחדל שנשלח בטעות כמפתח.
- רשימות המדדים נטענות מ-FMP:
  - `sp500-constituent`
  - `nasdaq-constituent`
  - `dowjones-constituent`

## העלאה ל-GitHub

העלה את כל הקבצים שבתיקייה זו ל-root של ה-repository והחלף את הקבצים הישנים.

## Streamlit Secrets

מומלץ לשמור את המפתח ב-Streamlit:

```toml
FMP_API_KEY = "המפתח-האמיתי-שלך"
```

אין לשתף את המפתח בצ'אט או ב-GitHub.

## מגבלת החשבון החינמי

בחשבון FMP חינמי קיימת מכסת קריאות יומית. הסריקה הפונדמנטלית דורשת מספר קריאות לכל מניה, ולכן v0.5 מגבילה את מספר המניות בכל הרצה. לאחר אימות המנוע נוכל לבחון שימוש ב-endpoints מסוג Bulk או חבילה מתאימה כדי לסרוק את כל המדדים בהרצה אחת.

## הערה לגבי MA50

FMP מספק את ערכי MA50 ו-MA200 הנוכחיים ב-Batch Quote. בגרסה זו `MA50 Rising` מיושם בשקיפות כ-`MA50 > MA200`, כלומר מגמה בינונית חיובית. חישוב שיפוע MA50 אמיתי ידרוש סדרת מחירים היסטורית נוספת.
